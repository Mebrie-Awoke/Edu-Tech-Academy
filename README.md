EduTech_Academy — README

This folder contains a small 4-page site for EduTech Academy:
- index.html
- about.html
- courses.html
- contact.html
- style.css

How to use:
1. Place all files in a folder named "EduTech_Academy".
2. Open any HTML file in your browser (double-click) — it works locally (no server required).
3. To create a zip: compress the "EduTech_Academy" folder into EduTech_Academy.zip.
